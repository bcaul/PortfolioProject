MNIST Classifier from Scratch

This project implements a feedforward neural network to classify handwritten digits from the MNIST dataset. It consists of:

  • A custom neural network class, which creates the neural network from scratch only using some clever linear algebra rather than using a deep learning framework.

  • A script that uses the custom neural network class and imports weights form a json file to classify the hand-drawn digits.

Right now the model is at 85% accuracy, although a way higher accuracy could be achieved with a smarter model.
