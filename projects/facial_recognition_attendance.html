Facial Recognition Attendance System

A real-time face recognition attendance system built with OpenCV and the face_recognition library. It detects and recognizes faces from a webcam feed, compares them with stored images, and logs attendance in a CSV file with timestamps.

How it works:

Basic.py:
• loads images and converts them into from BGR into RGB (which is needed for the facial_recognition library).
• Faces are found and encoded into numerical representations. The face encodings are then compared decided if the faces belong to the same person aswell as giving a similarity score.

Attendance.py:
• Attendance.py extends Basic.py into a real-time application by using a webcam to detect, recognise and log attendance.
